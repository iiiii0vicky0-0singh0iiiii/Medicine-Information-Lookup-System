# Medicine Lookup System

A web-based medicine lookup system that retrieves drug information from trusted public medical data sources and displays it in a user-friendly web interface.

##  Features

- Search medicines by name
- View detailed drug information
- Real-time data retrieval using external APIs
- Clean and responsive web interface
- Easy to use and lightweight

## 🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- API: Public medical data API
- Server: Apache (XAMPP / Localhost)
